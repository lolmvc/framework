    Autoloader implements the PHP Framework Interoperability Group's PHP Standards Recommendation for autoloading classes.

    http://github.com/php-fig/fig-standards/blob/master/accepted/PSR-0.md

    // Example which loads classes for the Exegesis annotation parser.
<pre><code>    $classLoader = new Autoloader('MattRWallace', ['vendor/matt/src','/vendor/matt']);
$classLoader->register();
</code></pre>
