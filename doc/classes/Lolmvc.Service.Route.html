Determines which controller is requested in the URI and creates an instance of it.

The remainder of the URI (Action and params) are passed to the controller's constructor so that the constructor can determine the appropriate actions.

Example:

The following code is how the App object handles routing and status 404 responses.

try {
   $router = new \Lolmvc\Service\Router($_SERVER['REQUEST_URI'], 'Skel');
} catch (\Lolmvc\Service\PageNotFoundException $e) {
   // get the error404 classname
   if (CUSTOM_404)
      $error404namespace = $appname;
   else
      $error404namespace = "lolmvc";

   $request = "error404";
   $message = $e->getMessage();
   if (!empty($message))
      $request .= "/$message/";
   $router = new
   \Lolmvc\Service\Route($request, $error404namespace);
}
